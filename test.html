Bottle Dataset Project

Author: Julio Gonzalez

1. Overview

This project uses a bottle detection dataset (images + annotations) for computer vision tasks. The dataset includes labeled bottles, bounding boxes, and supporting scripts. This website presents the structure, methodology, and results — without including the actual dataset files.

2. Repository Structure

Bottle_Dataset/ ├── data/ │ ├── raw/ # Original images (not included here) │ ├── processed/ # Preprocessed samples ├── annotations/ # COCO/CSV/XML annotations ├── scripts/ # Preprocessing + training scripts ├── models/ # Saved model checkpoints └── README.md

3. Methods

Images were annotated using bounding boxes. A preprocessing script cleaned, resized, and prepared the dataset for training. A convolutional neural network (such as AlexNet, ResNet18, or VGG16) was used to train on the annotated bottle examples.
